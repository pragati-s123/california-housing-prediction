# California Housing Price Prediction

A machine learning project focused on data preprocessing and pipeline building
using the California Housing dataset.

## Project Structure
- `data/` - dataset
- `notebooks/` - step by step exploration notebooks
- `main.py` - final pipeline script

## Topics Covered
- Exploratory data analysis
- Data visualization
- Handling missing values
- Feature scaling
- Categorical encoding
- Sklearn Pipelines

## Libraries Used
pandas, numpy, scikit-learn, matplotlib, seaborn

## Note
Learning project based on Hands-On Machine Learning by Aurélien Géron.
